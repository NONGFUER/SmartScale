# SmartScale 版本记录

> 当前发布包：`smartscale-2.13.3.24.tar.gz`（主版本 2.13.3，构建号 24，约 2026-07-24）
>
> **版本号规则**：主版本号由 `CMakeLists.txt` 的 `project(SmartScale VERSION 2.13.3)` 定义；
> 构建号通过 `cmake -DBUILD_NUMBER=N` 手动指定（默认 9），编译日期自动取当天。
> 版本号经 `src/version.h.in` 模板由 `configure_file` 生成到 `build/generated/version.h`，
> 由 `SystemInfoService` 暴露为 `appVersion` 并在状态栏/设置页显示。
>
> 项目采用手动版本号体系，源码主版本长期维持在 2.13.x，每次构建通过 `BUILD_NUMBER` 递增区分。
> 以下按开发阶段记录主要变更（按时间倒序，最新在前）。

---

## 2026-07-21 ~ 2026-07-24 系统设置改版与体验收尾

- **设置页改版**：设置对话框「公司信息」卡片改版，新增公众号二维码并重排为三列布局；增大空状态文字字号并统一颜色。
- **系统设置增强**：新增内存信息显示（`SystemInfoService` 解析 `/proc/meminfo`，<3GB 显示 "2GB" 否则 "4GB"），优化设置界面与 TTS 引擎。

## 2026-07-19 ~ 2026-07-22 虚拟键盘、4G 与网络能力完善

- **虚拟键盘**：新增自定义纯白明亮键盘风格（`src/ui/vkbdstyle/light/style.qml`），放大字号、优化交互，改为悬浮覆盖模式避免键盘遮挡输入框；键盘固定置于 `Overlay.overlay` 层（`z:99999`，`InputPanel.scale = 0.62`）。
- **四模式网络控制**：新增 `WifiOnly / CellularOnly / AllWifiPriority / AllCellularPriority` 四模式（默认 `AllCellularPriority`），基于 route-metric 实现优先级（优先 10 / 非优先 300），持久化到 `AppSettings.networkMode`，开机 5s 后恢复。
- **4G 能力**：4G 弹窗重写，支持开关重启记忆；`NetworkManagerService` 新增 `ip a` 快速路径（`refreshCellularStatusFast`，300~6000ms 快速轮询），以「接口有 IPv4 inet 地址且管理态 UP」判定数据激活，解决开关状态刷新慢问题。
- **WiFi/运营商**：新增 WiFi/自动切换设置与运营商中文显示；统一状态栏 WiFi 图标格数与 `WifiListDialog` 数据源（统一取自 `availableNetworks` 中匹配 SSID 的 `signal`）。
- **状态栏**：增大状态栏图标尺寸，更新信号与 WiFi 图标资源。

## 2026-07-18 ~ 2026-07-23 语音播报引擎重构（sherpa-onnx Matcha）

- **引擎重构**：语音播报引擎由 `piper + aplay` 管道重构为 **sherpa-onnx C API + Matcha 中文模型**（进程内合成，`VoiceSpeaker` 类名与 QML 接口不变），用 `dlopen(RTLD_LOCAL)` 隔离 onnxruntime 符号冲突。
- **模型切换**：先替换小雅(piper)模型为 Matcha-TTS，再整体重构为 sherpa-onnx Matcha 推理（`matcha-icefall-zh-baker`，22050Hz）。
- **性能与体验**：`NumberPad` 添加按键语音播报；保存成功语音播报与倒计时弹窗；预初始化 TTS 引擎消除首次播报延迟；多次调参 TTS 语速（最终 `genCfg.speed = 0.898`，相对最初调快约 10%）。
- **行为修正**：移除语音播报前缀；食材播报格式统一为 `！！！<中文食材名>！！！`（`FoodTranslator` 翻译后经 `CameraController::speakPredictedLabel` 播报）。

## 2026-07-14 ~ 2026-07-18 工作台交互与保存流程整合

- **保存流程重构**：新增保存确认弹窗（含价格输入与重复称重检测），金额单位统一为元/kg（`amount = unitPrice × netWeight`，`qRound(price*100)/100`）；DB 写入即上传并立即关闭遮罩；乐观保存优化。
- **工作台/状态栏**：新增底部状态栏并修复头像、边距、日期居中；重构工作台布局与用户信息展示，显示客户名称；统一全局字体（内嵌 PingFang SC）并重构页面布局；统一右侧区域卡片样式。
- **弹窗体系**：统一弹窗字号、返回按钮（`back2` 图标 + 文字）、遮罩与交互；`AlertDialog` 通用错误弹窗，错误提示脱敏（隐藏 URL 与技术细节）；修复键盘弹出遮挡（上移避让）与自动聚焦问题。
- **历史与详情**：新增称重记录表格弹窗、日历组件与日期筛选；重构称重记录查询弹窗、记录详情弹窗导航、图片查看弹窗（深色沉浸式）；修复金额计算、品类弹窗默认选中与遮罩覆盖状态栏。
- **识别**：支持识别接口多候选结果展示，推荐候选复用图片卡片并优先展示；智能识别按钮防频繁点击锁定机制。
- **认证**：未登录禁止网络请求并统一登录提示遮罩；拦截未登录状态下的识别操作；状态栏用户区域点击弹出退出确认。

## 2026-07-11 ~ 2026-07-13 全局样式与图片查看器

- 统一全局字体并重构工作台页面布局；统一右侧区域卡片样式。
- 图片查看器重构：白色卡片风格 → 顶部导航栏布局 → 深色沉浸式风格；增大全局字号并前置副摄像头视图。
- 食材品类改为两级联动，修复选中标签不显示；复用已缓存图片避免重复下载；修复 S3 预签名 URL 因附加 Bearer 头导致图片缓存 403。
- 登录后三个请求改为串行调度；修正秤精度单位为千分号。

## 2026-07-02 ~ 2026-07-10 登录认证、MQTT 上报与设备信息

- **认证增强**：实现无感刷新协调器（`AuthService` 全局锁 `m_isRefreshing` + `tokenRefreshCompleted`）与记住登录（存 `~/.config/SmartScale/last_login.conf`，base64 编码）；称重记录撤回（软删除）功能；替换硬编码序列号为动态读取，增加密码可见性切换。
- **MQTT**：新增 MQTT 客户端服务并接入设备信息上报、设备心跳上报、下行命令监听与告警上报；扩展设备信息上报字段并新增日志落盘。
- **调制解调器**：新增 IMSI 查询及 SIM 卡号 / IMSI 设备信息展示；修正 ICCID AT 命令并集成蜂窝模组服务；ML307B 网卡与串口共存脚本。
- **UI/素材**：用户头像功能；食材图片下载缓存与选择界面展示；食材品类全量拉取缓存；修复 WiFi 断开时 parent 解析错误导致白屏崩溃；增强 WiFi/4G 双模式兼容与动态设备发现。
- **手写输入**：集成中文手写输入与拼音切换功能（`QtMqtt` 第三方库加入 `3rdparty`）。

## 2026-06-28 ~ 2026-07-01 网络控制与品类弹窗

- 新增 Wi-Fi + 4G 网络控制模块；增强 WiFi/4G 双模式兼容与动态设备发现；修正 5G 标签判断逻辑。
- 修复 WiFi 断开后状态回退及页面渲染崩溃；修复 WiFi 断开时 parent 解析错误导致白屏崩溃。
- 重绘 WiFi 图标并放大状态栏与工位页元素；调整虚拟键盘与系统信息弹窗样式。
- 新增食材弹窗组件、搜索图标与添加食材按钮；实现创建食材 API 与设置弹窗。
- 重构记录详情弹窗为纯图片浏览模式；修复退出弹窗圆角并支持记录滑动切换。

## 2026-06-20 ~ 2026-06-27 品类、水印与存储

- **品类体系**：`CategoryCorrectionDialog` 实现本地食材搜索；修复遮挡虚拟键盘问题（改为非模态 + 外部遮罩，reparent 到 `window.contentItem`）；`ActionButton` 组件提取复用；食材卡片渐变立体感与上浮效果；系统信息弹窗遮罩修复。
- **水印/存证**：替换左下角公章为盾牌锁 + 存证哈希卡片；启动时读取并缓存设备序列号 SN；修复数据库路径。
- **业务**：集成 USER 域双接口与食材服务；手动品类选择模式；品类服务与选择弹窗重构；称重历史改为手动 AI 识别；食材翻译器去硬编码改为 API + 缓存；图片上传 CustId/RecoId 从 URL 查询移至 form-data；解耦拍照水印与 AI 识别并新增图片上传。
- **稳定性**：修复 userId 类型为 qint64 防止溢出；修复雪花 ID 溢出、refreshToken 格式及 Repo 查询空结果等关键缺陷；修复 bill 负值。
