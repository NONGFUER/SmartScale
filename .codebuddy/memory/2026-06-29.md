# 2026-06-29

## CellularDialog 4G QML 渲染问题修复
- **C++ (NetworkManagerService)**: `lastError()` 从 `Q_INVOKABLE` 改为 `Q_PROPERTY` + `NOTIFY lastErrorChanged`，新增 `setLastError()` setter 统一处理 23 处赋值+信号发射。修复 QML 绑定无法追踪 `lastError` 变化的问题。
- **QML (CellularDialog.qml)**: Switch 的 `checked` 属性从 `property bool checked: isSwitchChecked()`（只求值一次）改为直接绑定 `NetworkManager.cellularStatus` 范围比较，确保状态变化时开关 UI 自动同步。

## WorkstationPage.qml anchors 警告
- 第596行 MouseArea 在 RowLayout 内使用 anchors.fill，属于"layout-managed item 上用 anchors"的未定义行为。
- 解决方案：用 Item 包裹 ColumnLayout + MouseArea 作为 RowLayout 唯一子元素（用户尚未确认是否修改）。
