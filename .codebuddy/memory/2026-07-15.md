## 图片圆角 + 加高修复
- **问题1（无圆角）**：`Rectangle.clip:true` 只裁矩形包围盒，**不跟随 radius** → Image 四角仍是直角盖住灰瓦圆角。Qt6.8 实测确认。
- **修复**：改用 `MultiEffect` mask 方案（Qt6 标准）：
  - 源 `Image{visible:false}` + 遮罩 `Rectangle{id:imgMask; radius:12; color:"#FFFFFF"; visible:false; layer.enabled:true}` + 显示项 `MultiEffect{source:foodImg; maskEnabled:true; maskSource:imgMask; anchors.fill:foodImg}`
  - 关键：source 与 maskSource 都 `visible:false`，mask 须 `layer.enabled:true` 才能渲染成纹理；mask 用 alpha 通道，白色圆角矩形=可见区
  - 删除 imageContainer 的 `clip:true`（mask 已处理圆角，clip 反而在 hover 缩放时切出直角）
  - hover `scale` 从 Image 移到 imageContainer（整瓦缩放，圆角随之缩放不露直角）
- **问题2（太矮）**：`cellHeight 160→200`，图片高 `cellHeight-70` 由 90→130(+44%)，卡片总高 192 仍 < cellHeight 200 留 8px 行距

## 二次加高 + 加大卡片间距
- `cellHeight 200→250`；图片高 `cellHeight-70→cellHeight-80` = 170（+40，且增长慢于 cellHeight 以腾出间距）
- 卡片宽 `cellWidth-12→cellWidth-18`（水平间距 12→18）
- 间距公式：垂直 gap = cellHeight - (image + spacing12 + name~30 + margins20) = cellHeight - image - 62 = 250-170-62 = 18
- 结果：图片 170px、水平/垂直间距均 18px
