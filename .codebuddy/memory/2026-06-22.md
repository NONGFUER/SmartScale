## WorkstationPage.qml 称重组件贴底调整
- 右侧整列为 `ColumnLayout`（约 450 行起），子项默认顶部堆叠，底部留空白。
- 在名称卡片闭合（原 606 行 `}`）与"称重克数"标签之间插入弹性占位 `Item { Layout.fillWidth: true; Layout.fillHeight: true }`，吸收剩余高度，把称重标签+卡片整体挤到列底，紧贴 `mainCard` 下沿。
- 方案一（只改位置不改尺寸），未触动称重卡片高度。
- 后续发现弹性占位仍不生效，根因：中部区 `ColumnLayout`（约 524 行）使用了 `anchors.centerIn: parent`，导致它只是悬浮居中、未撑满父 `Item`，内部 `Layout.fillHeight` 吸收不到剩余空间。改为 `anchors.fill: parent` 后弹簧才真正生效，称重卡片贴到列底。教训：QML 中 `anchors.centerIn` 与子项 `Layout.fillHeight` 配合无效，要让 Layout 弹簧生效，容器必须 `anchors.fill` 撑满父级。
